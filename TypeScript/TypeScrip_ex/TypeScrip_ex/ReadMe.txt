#How to create Angularjs2 with Typescript based application:
1. Need to install Node js
2. Create a directory 
3. Add 3 main files namely 
	- package.json (contains all packages to be loaded into the project)
	- systemjs.config.js (contains system related configuration)
	- tsconfig.json (contains compliation configuration)
4. Open Command prompt ,navigate to the directory and install dependency files using the following command
	> npm install
